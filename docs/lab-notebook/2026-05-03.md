# 2026-05-03

## B1: INT pulse capture verified
Configured LSM6DSOX hardware single-tap detector via tap_int_test.py.
Threshold 500 mg per axis, ODR 416 Hz, single-tap mode, INT routed to I1 pin.

Saleae Logic Pro 8 connected to sensor I1 pin (D0) and GND. Captured at
25 MS/s, trigger on rising edge, 5 second buffer, 12.75 s window.

Tapped breadboard 4-5 times. 4 TAP_DETECTED events reported by host
(all X-axis, mixed sign as expected for taps in both directions).
4-5 corresponding rising edges visible on Saleae D0. Pulse widths
narrow (well under 50 ms host polling interval). Edges clean.

Wire-level measurement chain validated end-to-end on the sensor-INT side.
