# Paper versions

This study targets two possible venues. Both versions share the same
data, figures, and findings; they differ only in length and depth.

## paper/letters/ — PRIMARY TARGET

**IEEE Sensors Letters.** Strict 4-page limit (two-column), references
restricted to a minimum of one column. Papers exceeding 4 pages are
automatically transferred to IEEE Sensors Journal (per the IEEE Sensors
Letters reviewer guide). Rapid-disclosure venue; the counterintuitive
headline (on-host inference faster than on-sensor) suits the format.

Body-text budget with 2 figures + 1 table + ~14 references: ~2,800-3,000
words. Built by compressing the journal version ~48%.

## paper/journal/ — FALLBACK + MASTER SOURCE

**IEEE Sensors Journal.** 8-page soft limit (overlength charges beyond).
This is the full-length version: complete methodology, all five
limitations, the multimodal-distribution discussion, and the full
"what pre-registration did and did not do" treatment.

Dual role: (1) the fallback submission if Sensors Letters declines, and
(2) the master from which the lean Letters version is carved. We edit
the journal version into a coherent full-length paper, then derive
Letters from it by cutting.

## paper/figures/ — SHARED

Figures are shared across both versions. The Letters version uses a
tight subset (2-3 figures); the journal version can use more.

## Section numbering

Both versions use the same Roman-numeral section scheme:
- §I   Introduction
- §II  Background and Related Work
- §III Experimental Setup
- §IV  Methodology and Statistical Treatment
- §V   Results
- §VI  Discussion
- §VII Conclusion

(File prefixes 02-11 reflect drafting order, not paper order. The
results table file 02-* is an artifact/working table, not a numbered
section.)
